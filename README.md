# Generador-de-Passwords
# El presente proyecto consiste en el desarrollo de una aplicación web capaz de generar contraseñas seguras de manera automática. La página fue diseñada tomando como referencia una maqueta visual proporcionada previamente.

El sistema permite al usuario personalizar la contraseña mediante diferentes opciones, tales como:

# Selección de longitud (mínimo 9 caracteres).
# Inclusión de letras mayúsculas.
# Inclusión de letras minúsculas.
# Inclusión de números.
3 Inclusión de símbolos especiales.

El generador combina los caracteres seleccionados y produce una contraseña aleatoria cada vez que el usuario presiona el botón "GENERAR". Además, el sistema incluye:

Indicador visual de nivel de seguridad (débil, media o fuerte).

Botón para copiar la contraseña al portapapeles.

Validación para asegurar que al menos un tipo de carácter esté seleccionado.
